## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run App

```bash
python3 ./src/main.py
```

## Документация Flet для разработки

- Тематические справочники: [docs/flet/README.md](docs/flet/README.md)
- Skill для агента: `.cursor/skills/flet/SKILL.md`
- Полный архив PDF: `docs/flet-docs.pdf`

В запросах к агенту указывайте `@docs/flet/<тема>.md` и `@src/<файл>.py`.
